## What Wavox needs

Wavox requires the following permissions to work properly:

| Permission | Why |
|------------|-----|
| Connect | Join voice channels |
| Speak | Play audio in voice channels |
| Send Messages | Respond to commands and show now playing info |
| Embed Links | Display rich embeds with track details |
| Add Reactions | React to now playing cards |
| Use Application Commands | Support slash commands |
